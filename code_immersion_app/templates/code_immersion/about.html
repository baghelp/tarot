About Code Immersion

I wanted to have an interface that would keep track of my progress while I rewrite an existing codebase. It's possible something already existed, but this seemed like a fun project to build myself. Right now, it just includes some code examples for C. I'm still working on integrating:

  • saving progress
  • importing arbitrary codebases
In this project, I used the following tools:
  • CodeMirror, which uses the MIT license -- see here
  • jsdiff, which uses the BSD license -- see here
  • django, which uses the BSD license -- see here
